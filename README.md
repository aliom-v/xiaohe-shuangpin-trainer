# 小鹤双拼练习器

一个功能丰富的小鹤双拼在线练习工具。

## 功能特性

- 🎹 可视化键盘，实时高亮下一个键位
- 🔊 打字音效
- 🌓 亮色/暗色主题
- 📱 移动端适配，支持触摸输入
- 📖 新手教程
- 🎯 专项练习（变位声母/复杂韵母/零声母）
- 💡 提示模式 / 🙈 盲打模式 / ⏱️ 限时挑战
- 📊 练习统计和错误分析
- 🌐 在线获取随机文本
- 🔄 自动下一段模式

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- pinyin-pro

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000
