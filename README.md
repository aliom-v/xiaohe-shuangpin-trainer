# 小鹤双拼练习器

一个简单直观的小鹤双拼在线练习工具。

**在线体验：** https://xiaohe-shuangpin-trainer.vercel.app

## 一键部署（Vercel）

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/aliom-v/xiaohe-shuangpin-trainer)

## 快速上手

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 规则提示

- 1-2 字母拼音可直接输入（开关可关），也可用对应双拼
- 3+ 字母按小鹤双拼规则输入（如 xue → xt）

## 快捷键

- `Space` 随机文本
- `Tab` 跳过当前字
- `Esc` 结束练习
- `Backspace` 退回输入

## 常用命令

- `npm run dev` 本地开发
- `npm test` 双拼映射检查
